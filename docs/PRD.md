# 마이링크 (MyLink) PRD (제품 요구사항 정의서)

## 1. 프로젝트 개요
- **프로젝트 명**: 마이링크 (MyLink)
- **목적**: 개인 인플루언서나 크리에이터가 자신의 여러 소셜 미디어 플랫폼과 콘텐츠를 하나의 단일 링크로 모아 팬들에게 간편하게 공유할 수 있는 통합 프로필 서비스 제공. **(전 기능 무료 제공)**
- **대상 사용자**:
  1. **개인 인플루언서 (User)**: 자신의 단일 링크를 생성하고 관리하는 주체
  2. **링크 방문자 (Visitor)**: 인플루언서가 공유한 링크에 접속하여 여러 콘텐츠를 확인하는 일반 대중
- **기술 스택 및 인프라**:
  - **데이터베이스/인증**: Firebase (Firestore Database, Firebase Auth)
  - **프론트엔드/디자인 시스템**: React (Next.js 권장) / **shadcn/ui** 디자인 시스템 활용
  - **Next.js 렌더링 전략 분리**: 
    - **방문자 화면(`/[username]`)**: 대중을 상대로 SEO 최적화와 즉각적인 로드 속도가 생명증명되므로 **ISR(점진적 정적 재생성)** 또는 **SSR** 전략 채택
    - **인플루언서 대시보드 관리자 화면**: SEO가 불필요하며 빠른 동적 상호작용이 필요한 인증 구역이므로 **CSR(클라이언트 사이드 렌더링)** 전략 채택

## 2. 필수 서비스 기능 목록
1. **간편 로그인**: 구글(Google) 소셜 로그인 전용
2. **고유 프로필 URL 생성**: 사용자 고유의 접속 가능한 단일 웹 사이트 주소 발급 (예: mylink.com/username)
3. **텍스트 기반 프로필 꾸미기**: 닉네임, 짧은 소개글(Bio) 텍스트 편집 및 기본 테마 설정
4. **링크 관리 (CRUD)**: 링크 추가, 파비콘 자동 로드(실패 시 기본 아이콘 대체), 수정, 숨김, 삭제 및 순서 변경(드래그 앤 드롭)
5. **방문자 및 클릭 통계 (Analytics)**: 프로필 전반의 단순 조회수(PV) 및 개별 버튼 클릭수 통계 (기간 필터 지원)
6. **SEO 및 Open Graph(OG) 지원**: 공유 시 노출될 썸네일 이미지 동적 생성 지원

## 3. 기능 개별 명세 및 정책 가이드

### 3.1. 인플루언서 (사용자) 기능 정책
- **소셜 로그인 및 정책 동의**
  - 가입/로그인은 구글 계정(OAuth) 전용으로 처리합니다.
  - 가입 화면 하단에 **이용약관 및 개인정보 처리방침** 링크를 명시하여 사용자 동의를 득하는 동적 흐름을 구축합니다.

- **고유 프로필 URL (Username) 생성 규칙**
  - **문자열 규칙**: **영문 소문자, 숫자, 하이픈(-), 언더바(_)** 조합만 허용합니다.
  - **길이 제한**: 최소 **3자** 이상, 최대 **15자** 이하로 제한합니다.
  - **블랙리스트 필터링**: 시스템 예약어(login, admin, terms 등) 및 비속어를 사전에 차단하기 위한 필터링 우회 로직을 도입합니다.

- **텍스트 글자 수 제한 및 테마 설정**
  - **닉네임(디스플레이 네임)**: 최대 30자 이내
  - **자기소개(Bio)**: 최대 2줄 텍스트로 **80자(영문 기준 160 bytes) 이내**로 제한하며, 초과 시 말줄임표(`...`) 처리합니다.

- **링크 등록 시 예외 처리 (Edge Cases)**
  - **URL 제목**: 최대 **50자** 이내로 제한합니다.
  - **파비콘 추출 실패(Fallback)**: 사용자가 입력한 URL의 파비콘 렌더가 불가능(CORS, 미존재 등)할 경우, 브라우저에 엑스박스가 뜨지 않고 서비스 내장 **기본 아이콘(Default Link Icon)**으로 즉각 갈아끼웁니다.
  - **악성 링크 필터링**: 등록한 웹사이트가 피싱이나 불법 콘텐츠 사이트인지 식별하기 위해 Google Safe Browsing API 검증 등을 연동해, 문제 발생 시 강제 비활성화 차단 조치를 도입합니다.

- **통계 (Analytics) 기준**
  - 기본적으로 트래픽 로드를 감안하여 '단순 새로고침을 포함시킨 조회 수(PV)'를 노출합니다. (향후 IP/기기 검증을 통한 UV로 스왑 가능성 검토)
  - 대시보드 화면에 `최근 7일`, `최근 30일`, `전체 기간` 옵션 필터를 제공하여 기간별 성과 분석 기능을 지원합니다.

- **SEO 및 공유 썸네일 (Open Graph)**
  - 이미지 업로드 기능이 없으므로, 인플루언서가 링크 공유 시 **[선택한 테마 배경색 + 중앙 닉네임 문자열]**을 합성 조합한 이미지를 동적 렌더링(예: Vercel OG 활용)하여 썸네일로 자동 제공합니다.

### 3.2. 링크 방문자 (Visitor) 경험 정책
- 모바일(인스타그램 인앱 브라우저 등) 기반으로 즉시 화면이 나타나도록 가볍고 빠르게 빌드합니다.
- 방문자 페이지 최하단(Footer)에 작게 '마이링크' 로고 및 **이용약관/개인정보 처리방침** 링크를 위치시킵니다.
